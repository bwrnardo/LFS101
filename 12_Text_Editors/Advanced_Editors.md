# `vi` and `emacs`

- Both can run on a non graphical environment and a graphical one

## Introduction to `vi`

- Standard tool installed virtually on all distributions
- `VIM` - **V**i **IM**proved
- `gvim` is **GNOME`s** graphical interface
- `kvim` is **KDE`s** graphical interface
- `vi` doesnt need the mouse to use

### `vimtutor`

- Typing `vimtutor` gives a very extense tutorial on how to use `vim`

## Introduction to `emacs`

- `emacs` is highly customizable and includes a number of features
- Can be used for email, debugging, etc...
- Uses the **CTRL** and Meta (**Alt** or **Esc**) for special commands
